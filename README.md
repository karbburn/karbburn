# Things I've Built

### 📈 [BondFactor](https://bondfactor.vercel.app)

An interactive playground for the Indian Government Securities yield curve.

Experiment with **parallel shifts**, **twists**, **steepeners**, **flatteners**, and **butterfly scenarios** instead of just reading about them.

**Python · Next.js · Quantitative Finance**

**→ Repository:** https://github.com/karbburn/BondFactor

---

### 📊 [CorrShift](https://corrshift.vercel.app)

Markets rarely move independently.

CorrShift tracks rolling correlations across multiple financial assets and highlights when those relationships begin to change.

**Python · FastAPI · Next.js · Pandas**

**→ Repository:** https://github.com/karbburn/correlations-anomaly-detector

---

### 📉 [Factor Exposure Analyzer](https://factor-analyzer.vercel.app)

A factor analytics tool for understanding what actually drives portfolio returns.

Estimate exposure to **Market**, **Value**, **Momentum**, **Size**, and **Volatility** using rolling regressions.

**Python · FastAPI · Statsmodels**

**→ Repository:** https://github.com/karbburn/Factor-Exposure-Analyzer

---

### 💊 [DistrictDx](https://districtdx.vercel.app)

A data-driven framework ranking **785 Indian districts** by pharmaceutical market attractiveness.

Built for **Trilytics 2026 (IIM Calcutta PGDBA Conclave)**.

**Next.js · Python · Data Analytics**

**→ Repository:** https://github.com/karbburn/DistrictDx
